# Set

It's an array with unique value

## Create new

```javascript
const mySet = new Set();
```

## Add elements

```javascript
mySet.add(myVariable);
```

## Count elements

```javascript
mySet.size;
```

## Check if element exists

```javascript
mySet.has(myVariable);
```

## Delete an element

```javascript
mySet.delete(myVariable);
```

## Clear all elements

```javascript
mySet.clear();
```
